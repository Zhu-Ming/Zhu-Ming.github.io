LeetCode-11盛水最多的容器

2019-11-03
LeetCode
Word count: 716   |   Reading time≈ 2 min

题目：

给定 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂

直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以

容纳最多的水。

说明：你不能倾斜容器，且 n 的值至少为 2。

图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大

值为 49。


示例:

输入: [1,8,6,2,5,4,8,3,7]

输出: 49

下图是此数组对应的图形，以及它的最大面积。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/container-with-most-water

##思路1：
###暴力解法（优化后的）

选定长方形的长，然后依据长来从头开始找数组中的相距长度为此的两个数值，取出它们的最小值，然后求面

积，如果面积比上一个值大，就修改上一个面积值，使得面积永远是最大值。

代码块：

public int maxArea(int[] height) {
    int area=0;
    int h=0;
    for(int i=1;i<height.length;i++) {//根据不同的长来查找
        for(int a=0;a<height.length;a++) {
            int b=a+i;
            if(b>height.length-1) {
                break;
            }else {
                h=height[a]>height[b]?height[b]:height[a];
                if(h*i>area) {
                    area=h*i;
                }
            }
        }

    }
    return area;
}

复杂度分析

时间复杂度：O(n^2)，计算所有 n(n−1)/2种高度组合的面积。

空间复杂度：O(1)，使用恒定的额外空间。

##思路二
###双指针法（最优解法）

首先，定义两个变量分别指向数组的头和尾下标，然后比较它们对应的数组中的值，取小的数值与它们之间的
距离相乘，这就是当前的面积，如果当前面积比上一个面积大就修改面积变量值，使其永远存放最大面积值。

####分析双指针法：

因为此思路是根据长方形的面积始终由它的最小的宽决定的，所以从两头开始遍历，改变指向小数值的那个变
量使其指向下一个（对于前面的来说是后一个，对于后面的来说是前一个），（为什么这么做呢?因为改变了小
的那个变量的数值使两个变量之间对应的数组值中即使有比它还要小的，那么这个值与其他数组值形成的面积
也绝对不会比上一个大）

代码块：

public int maxArea(int[] height) {
    int area=0;
    int head=0,last=height.length-1;
    int h=0;
        while(head<last) {
            h=height[head]>height[last]?height[last--]:height[head++];
            if(h*(last-head+1)>area) {
                area=h*(last-head+1);
            }
        }
    return area;
}

时间复杂度：O(n)

空间复杂度：O(1)
