            printf(" %c",s->data[i]);

        return 0;

两者区别：

静态分配时就是数组，长度一定，必须在程序运行前指定，而动态分配则不用事先指定长度，但它要使用malloc函数，动态申请一个数组空间，对此空间还是使用下标访问每一个元素。还有动态分配时要指定最大空间长度（应该也可以不用指定，不过如果结构体中有，那么函数中就会用到，所以一定要指定）。其他大致相同。
