LeetCode-25k个一组翻转链表

2019-11-16
LeetCode
Word count: 579   |   Reading time≈ 2 min

题目：

给你一个链表，每 k 个节点一组进行翻转，请你返回翻转后的链表。

k 是一个正整数，它的值小于或等于链表的长度。

如果节点总数不是 k 的整数倍，那么请将最后剩余的节点保持原有顺序。

示例 :

给定这个链表：1->2->3->4->5

当 k = 2 时，应当返回: 2->1->4->3->5

当 k = 3 时，应当返回: 3->2->1->4->5

说明 :

######你的算法只能使用常数的额外空间。
######你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

思路：递归把一个链表看成n个长度为k的链表（不足的也看成一段），然后从1到n段进行逆序排列，将第i段的next指向第i+1段的逆序结果。

代码：

public ListNode reverseKGroup(ListNode head, int k) {
    if(head==null)//如果恰好就是n段（即每一段都是k个），那么最后向下递归时，head会为null
        return null;
    ListNode res=head;
    ArrayList<ListNode> lists = new ArrayList<ListNode>();
    for(int i=0;i<k&&head!=null;i++) {//先把k个结点存下来
        lists.add(head);
        head=head.next;
    }
    int len=lists.size();
    if(len==k) {//如果刚好是k个结点改变为逆序
        ListNode result=res=lists.get(len-1);//将此段链表最后一个结点为头结点，用作返回值。
        for(int i=len-2;i>-1;i--) {
            res.next=lists.get(i);
            res=res.next;
        }
        res.next=reverseKGroup(head, k);//将此段链表的结尾连接上下一段链表的逆序。
        return result;
    }else {//不是k个结点不改变原来的顺序
        return res;
    }
}

思路：思想与上一个差不多，就是在处理逆序的方式不同。

代码：

public ListNode reverseKGroup(ListNode head, int k) {
    if (head == null) return null;
    if (k <= 0) return null;
    if (k == 1) return head;
    ListNode toHead = new ListNode(0);
    toHead.next = head;
    ListNode temp = toHead;
    while (temp.next != null) {
        temp = reverseSingleK(temp, k);
    }
    return toHead.next;
}

public ListNode reverseSingleK(ListNode before, int k) {
    ListNode pre = before.next;
    ListNode cur = pre.next;
    int flag = 0;
    for (int i = 1; i < k; i ++) {
        if (cur == null) {
            flag = i;
            break;
        }
        ListNode temp = cur.next;
        cur.next = pre;
        pre = cur;
        cur = temp;
    }
    if (flag != 0) {
        ListNode ret = pre;
        cur = pre.next;
        for (int i = 2; i < flag; i ++) {
            ListNode temp = cur.next;
            cur.next = pre;
            pre = cur;
            cur = temp;
        }
        ret.next = null;
        return ret;
    } else {
        ListNode ret = before.next;
        before.next = pre;
        ret.next = cur;
        return ret;
    }
}
