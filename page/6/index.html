LeetCode-18四数之和

2019-11-09
LeetCode

题目：

给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 
d ，使得 a + b + c+ d 的值与 target 相等？找出所有满足条件且不重复的四元组。

 注意：
 答案中不可以包含重复的四元组。

 示例：
 给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。

 满足要求的四元组集合为： [ [-1, 0, 0, 1], [-2, -1, 1, 2], [-2, 0, 0, 2] ]

 来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/4sum
 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路：
双指针选定法，首先对数组进行排序，然后定义一个指向最左与最右的指针，然后从这两个指针中间选取两个值构造四数数组，看看他们的和合不合适，合适就加到数组列表中，不合适就变换中间的两个值。注意此时可以用一些剪枝方法出去不合适的四个数字。

代码：

    public List<List<Integer>> fourSum(int[] nums, int target) {
        List<List<Integer>> list = new ArrayList<>();
        if (nums.length < 4) {//剪枝方法
            return list;
        } else {
            Arrays.sort(nums);
            int l = 0, r = nums.length - 1, lr = 0, rl = 0;
            for (l = 0; l < nums.length - 3; l++) {
                if (l > 0 && nums[l] == nums[l - 1]) {
                    continue;
                }
                for (r = nums.length - 1; r >= l +3; r--) {
                    if (r < nums.length - 1 && nums[r] == nums[r + 1]) {//剪枝方法
                        continue;
                    }
                    lr = l + 1;//开始指针的右边一个指针
                    rl = r - 1;//最右边一个指针的左边一个指针
                    while (lr < rl) {
                        int value = nums[l] + nums[r] + nums[lr] + nums[rl];
                        if (value == target) {
                            list.add(Arrays.asList(nums[l], nums[lr], nums[rl], nums[r]));
                            while (lr < rl && nums[lr] == nums[lr + 1])//剪枝方法
                                lr++;
                            while (lr < rl && nums[rl] == nums[rl - 1])//剪枝方法
                                rl--;
                            lr++;
                            rl--;
                        } else if (value > target) {
                            rl--;
                        } else {
                            lr++;
                        }
                    }
                }
            }
        }
        return list;


}

思路二：也是默认选定两个是定的，不过此处是将左边两个先选定，然后依照在选定后面两个，不断缩小后面两个的距离，注意这里也要用到跟多的剪枝方法。

代码：

public List<List<Integer>> fourSum(int []nums,int target){
    int len=nums.length;
    List<List<Integer>>list=new ArrayList<>();
    if(len<4) 
    {
        return list;
    }
    int value;
    Arrays.sort(nums);
    for(int i=0;i<len-3;i++) 
    {
        if(i>0&&nums[i-1]==nums[i])//如果这一个跟上一个一样就没必要再去取这一个了
        {
            continue;
        }
        value=nums[i]+nums[i+1]+nums[i+2]+nums[i+3];//如果以nums[i]开头的数组的最小的和都大于target，说明i以后的数字之和都大于target。
        if(value>target)
        {
            break;
        }
        value=nums[i]+nums[len-1]+nums[len-2]+nums[len-3];
        if(value<target)//如果以nums[i]开头的数组中的最大值都小于target，则说明i所有的数组之和都小于target
        {
            continue;
        }
        for(int j=i+1;j<len-2;j++)
        {
            if(j>i+1&&nums[j]==nums[j-1])//意义同上一层
            {
                continue;
            }
            value=nums[i]+nums[j]+nums[j+1]+nums[j+2];
            if(value>target) {//意义同上一层
                break;
            }
            value=nums[i]+nums[j]+nums[len-1]+nums[len-2];
            if(value<target) {//意义同上一层
                continue;
            }
            int l=j+1;
            int r=len-1;
            while(r>l) {
                value=nums[i]+nums[j]+nums[l]+nums[r];
                if(value==target) {
                    list.add(Arrays.asList(nums[i],nums[j],nums[l],nums[r]));
                    while(l<r&&nums[l]==nums[l+1])l++;
                    while(l<r&&nums[r]==nums[r-1])r--;
                    l++;
                    r--;
                }else if(value>target) {
                    r--;
                }else {
                    l++;
                }
            }
        }
    }
    return list;
}

LeetCode-17电话号码的字母组合

2019-11-08
LeetCode

题目：

给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。

给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。

示例:

输入："23"
输出：["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"].
说明:
尽管上面的答案是按字典序排列的，但是你可以任意选择答案输出的顺序。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/letter-combinations-of-a-phone-number
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路：我的思路，每次排序都可以看为将上一次的结果与下一次的要找的字符进行排序，注意将上一次产生的每个结果看成一个整体，然后与后面的字符进行排列，比如“23”，情况下可以看成（a,b,c）与（d,e,f）进行排列，而“233”可以先看成（a,,b,c）与(d,e,f)进行排列，然后这一次排列的结果为（ad, ae, af, bd, be, bf, cd, ce, cf）在与（d,e,f）进行排列。

代码：

public List<String> letterCombinations(String digits) {
List<String>list=new ArrayList<>();
if(digits==null||digits.length()==0)
    return list;
    String nums[]=new String[10] ;
    nums[1]="";
    nums[2]="abc";
    nums[3]="def";
    nums[4]="ghi";
    nums[5]="jkl";
    nums[6]="mno";
    nums[7]="pqrs";
    nums[8]="tuv";
    nums[9]="wxyz";
    int value1=0;
    int pos=0;//首先要将第一个数字对应的字符串的每一个字符当作这一次的结果，为下一次进行准备。注意此处有特别情况当value1的值不同时初始化列表的长度也不一样。
    value1=digits.charAt(pos++)-'0';
    list.add(nums[value1].charAt(0)+"");
    list.add(nums[value1].charAt(1)+"");
    list.add(nums[value1].charAt(2)+"");
    if(value1==7||value1==9) {
        list.add(nums[value1].charAt(3)+"");
    }
    for(int i=pos;i<digits.length();i++) {
        value1=digits.charAt(i)-'0';
        List<String>result=new ArrayList<>();
        for(int j=0;j<list.size();j++) {
            for(int k=0;k<nums[value1].length();k++) {
                result.add(list.get(j)+nums[value1].charAt(k));//注意要用中间变量存储结果，不然如果在原变量的基础上，循环条件就一直在改变了。
            }
        }
        list=result;
    }
    return list;
}

    class Solution {
     List<String> list;
    String[] array={" ","","abc","def","ghi","jkl","mno","pqrs","tuv","wxyz"};
    public List<String> letterCombinations(String digits) {
      list=new ArrayList<>();
         if(digits.equals(""))
             return list;
        compose(digits,0,"");
        return list;
    }
    public void compose(String digits,int index,String str){
        if(index==digits.length())
        {
            list.add(str);
            return;
        }
        char c=digits.charAt(index);
        String letter=array[c-'0'];
        for(int i=0;i<letter.length();i++)
            compose(digits,index+1,str+letter.charAt(i));
        return;
    }
}

LeetCode-16最接近的三数之和

2019-11-07
LeetCode

题目：

给定一个包括 n 个整数的数组 nums 和 一个目标值 target。找
出 nums 中的三个整数，使得它们的和与 target 最接近。返回这三
个数的和。假定每组输入只存在唯一答案。

例如，给定数组 nums = [-1，2，1，-4], 和 target = 1.

与 target 最接近的三个数的和为 2. (-1 + 2 + 1 = 2).

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/3sum-closest
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路：这题我是没有想法的，好不容易写的，调试半天，结果竟然位列12%左右，那就直接上大佬的代码；

分析思路：

然后依据上一题的双指针算法，对数组排一下序，从当前位置出发，
记录下每三个数字的和，然后用它们的和与初始的和对比。如果比初始更接近target，就修改要作为返回结果的和。如果当前的和比target大说明减小一点值可能跟接近target，如果比target小，那么可能大一点的和会更接近target。注意此处要进行下一次的循环时
可以优化一下，跳过那些已经出现过的数值。

代码：

class Solution {
    public int threeSumClosest(int[] nums, int target) {
        Arrays.sort(nums);
        int begin,end,currSum,sum=nums[0]+nums[1]+nums[2];
        for(int i=0;i<nums.length-2;i++){
            begin=i+1;
            end=nums.length-1;
            while(begin<end){
            currSum=nums[begin]+nums[end]+nums[i];
            if(Math.abs(target-currSum)<Math.abs(target-sum)){
                sum=currSum;
                }
            if(currSum>target){//如果当前和小于target，那么跳过重复的部分。
                while(begin<end&&nums[end]==nums[end-1])end--;
                end--;
            }else if(currSum<target){//如果当前大于目标值，那么跳过重复的数字，继续下一次循环
                while(begin<end&&nums[begin]==nums[begin+1])begin++;
                begin++;
            }else{
                    return sum;//最接近了，直接返回
                }
            }    
        }
    return sum;
    }
}
