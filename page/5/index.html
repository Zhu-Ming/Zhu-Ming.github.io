LeetCode-11盛水最多的容器

2019-11-03

题目：

给定 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂

直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以

容纳最多的水。

说明：你不能倾斜容器，且 n 的值至少为 2。

图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大

值为 49。


示例:

输入: [1,8,6,2,5,4,8,3,7]

输出: 49

下图是此数组对应的图形，以及它的最大面积。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/container-with-most-water

##思路1：
###暴力解法（优化后的）

选定长方形的长，然后依据长来从头开始找数组中的相距长度为此的两个数值，取出它们的最小值，然后求面

积，如果面积比上一个值大，就修改上一个面积值，使得面积永远是最大值。

代码块：

public int maxArea(int[] height) {
    int area=0;
    int h=0;
    for(int i=1;i<height.length;i++) {//根据不同的长来查找
        for(int a=0;a<height.length;a++) {
            int b=a+i;
            if(b>height.length-1) {
                break;
            }else {
                h=height[a]>height[b]?height[b]:height[a];
                if(h*i>area) {
                    area=h*i;
                }
            }
        }

    }
    return area;
}

复杂度分析

时间复杂度：O(n^2)，计算所有 n(n−1)/2种高度组合的面积。

空间复杂度：O(1)，使用恒定的额外空间。

##思路二
###双指针法（最优解法）

首先，定义两个变量分别指向数组的头和尾下标，然后比较它们对应的数组中的值，取小的数值与它们之间的
距离相乘，这就是当前的面积，如果当前面积比上一个面积大就修改面积变量值，使其永远存放最大面积值。

####分析双指针法：

因为此思路是根据长方形的面积始终由它的最小的宽决定的，所以从两头开始遍历，改变指向小数值的那个变
量使其指向下一个（对于前面的来说是后一个，对于后面的来说是前一个），（为什么这么做呢?因为改变了小
的那个变量的数值使两个变量之间对应的数组值中即使有比它还要小的，那么这个值与其他数组值形成的面积
也绝对不会比上一个大）

代码块：

public int maxArea(int[] height) {
    int area=0;
    int head=0,last=height.length-1;
    int h=0;
        while(head<last) {
            h=height[head]>height[last]?height[last--]:height[head++];
            if(h*(last-head+1)>area) {
                area=h*(last-head+1);
            }
        }
    return area;
}

时间复杂度：O(n)

空间复杂度：O(1)

LeetCode-9回文数

2019-11-02

题目：

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例 1:

输入: 121
输出: true
示例 2:

输入: -121
输出: false
解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
示例 3:

输入: 10
输出: false
解释: 从右向左读, 为 01 。因此它不是一个回文数。

最优解法：将首先可以肯定数字为负数或者（末尾有0且不等于0）情况下其反转后一定是错的。0一定是对的。下面讨论数为正数，（正数的长度为偶数）只要正数的前一半的数值等于后一半的数值就可以了（正数的长度为奇数的话）只要后一半的数值等于前一半对10取整即可（因为中间的那一个不需要有对称的位置）。不过我这里将其改了一下使代码更加简单（运行时间与原代码相差不大）

代码：

public boolean isPalindrome(int x) {
    if (x < 0 || (x % 10 == 0 && x != 0)) return false;
        int reverseNum=0;
        int y=x;
        while(x!=0) {
            reverseNum=reverseNum*10+x%10;
            x=x/10;
        }
        return y==reverseNum;
}

我的解法：将一个字符串，数字小于0返回false，为0返回true，对于正数将他赋值到一个字符串里，然后分别定义两个变量从头，尾开始向中间遍历，如果他们所在位置的字符不同说明不对称（即无法成为回文结构）。

代码：

public boolean isPalindrome(int x) {
    if(x<0)
        return false;
    if(x==0)
        return true;
        String str=""+x;
        int i=0,j=str.length()-1;
        while(i<j) {
            if(str.charAt(i)!=str.charAt(j)) {
                return false;
            }else {
                i++;
                j--;
            }
        }
        return true;
}

总结：字符串的操作运行时间要多余直接对数值进行操作的时间，应尽量采用操作数值。

LeetCodeo0008

2019-11-01

题目：

请你来实现一个 atoi 函数，使其能将字符串转换成整数。

首先，该函数会根据需要丢弃无用的开头空格字符，直到寻找到第一个非空格的字符为止。//去除开头空格

当我们寻找到的第一个非空字符为正或者负号时，则将该符号与之后面尽可能多的连续数字组合起来，作为该整数

的正负号；//第一个字符是+-或非空字符，则取接在符号后的连续数字假如第一个非空字符是数字，则直接将其与

之后连续的数字字符组合起来，形成整数。该字符串除了有效的整数部分之后也可能会存在多余的字符，这些字符可

以被忽略，它们对于函数不应该造成影响。

注意：假如该字符串中的第一个非空格字符不是一个有效整数字符、字符串为空或字符串仅包含空白字符时，则你的

函数不需要进行转换。

在任何情况下，若函数不能进行有效的转换时，请返回 0。 说明： 假设我们的环境只能存储 32 位大小的有符号

整数，那么其数值范围为 [−2^31, 2^31 − 1]。如果数值超过这个范围，请返回 INT_MAX (2^31 − 1) 或 INT_MIN (−231) 。

示例 1: 输入: “42”输出: 42

示例 2: 输入: “ -42” 输出: -42

解释: 第一个非空白字符为 ‘-‘, 它是一个负号。

我们尽可能将负号与后面所有连续出现的数字组合起来，最后得到 -42 。

示例 3: 输入: “4193 with words” 输出: 4193 解释:

转换截止于数字 ‘3’ ，因为它的下一个字符不为数字。

示例 4: 输入: “words and 987” 输出: 0

解释: 第一个非空字符是 ‘w’,

但它不是数字或正、负号。 因此无法执行有效的转换。

示例 5: 输入: “-91283472332” 输出: -2147483648

解释: 数字

“-91283472332” 超过 32 位有符号整数范围。 因此返回 INT_MIN (−2^31) 。

思路：

一：检查输入的字符是否是空是返回0，首先对其去空，然后根据第一个字符的符号决定接下来的操作，如果不是非法字符就继续操作后面的字符找到为连续的数值的个数,然后调用函数将字符串转化为数值，如果发生了异常，且字符串中靠前的子字符串不能转化为数值就说明是非法字符导致的异常，如果靠前面的子字符串可以转化为数值就说明发生了越界，根据符号决定它是向上越界还是向下越界，如果是向上越界就返回整型的最大值，反之就返回整型的最小值。如果没有发生异常，那么最终返回字符串转化后的数值。

代码：

public int myAtoi(String str){
           if(str==null)
            {
                return 0;
            }else {
                str=str.trim();//如果此处不主动将str赋值为str.trim();leetCode上显示通不过去？不知道为什么？
                if(str.length()==0) {
                    return 0;
                }
                int index=0;
                for(int i=1;i<str.length();i++) {
                    if(str.charAt(i)<='9'&&str.charAt(i)>='0') {
                        index++;
                    }else {
                        break;
                    }
                }
                try {
                    return Integer.valueOf(str.substring(0, index+1));
                }catch(Exception e) {
                    int a;
                    try {
                        a=Integer.valueOf(str.substring(0, 3));
                    }catch(Exception e1){
                        return 0;
                    }
                    return a>0?Integer.MAX_VALUE:Integer.MIN_VALUE;
                }
            }
}

思路二：

将整个字符串当成一个字符数组，然后遍历，判断第一个字符是否合格，如果第一个字符是‘ ’那么继续向后取字符，如果为不合格字符就返回0，合格继续向后读取字符，并判断读取到的每个字符是否是数值，是就先判断如果加上去是否可能会越界，会就返回相应的整型顶值，如果不会越界，就将上一个结果*10+此字符表示的数值。如果读到了不是数值的字符且还没有发生越界，那么就带符号的它的值。

代码：

public static int Mytoai_2(String s) {
        int flag=1;
        boolean start=true;
        int result=0;
        for(int i=0;i<s.length();i++){
            if(start) {
                if(s.charAt(i)==' ') {
                    continue;
                }
                if(s.charAt(i)=='-') {
                    flag=-1;
                }else if(s.charAt(i)=='+') {
                    flag=1;
                }else if(Character.isDigit(s.charAt(i))) {
                    result=s.charAt(i)-'0';
                }else {
                    break;
                }
                start=false;//标记一下已读过第一个不是空格字符了，后就不在进入此判断体内。
            }else {
                if(Character.isDigit(s.charAt(i))) {
                    if(flag==1&&(result>Integer.MAX_VALUE/10||(result==Integer.MAX_VALUE/10&&s.charAt(i)>'7'))) {//整型的上限是2^31-1
                        return Integer.MAX_VALUE;
                    }
                    if((flag==-1)&&(result>Integer.MAX_VALUE/10||(result==Integer.MAX_VALUE/10&&s.charAt(i)>'8'))) {//此处任然使用正数判断只是范围变化了而已。
                        return Integer.MIN_VALUE;
                    }
                    result=result*10+s.charAt(i)-'0';
                }else {
                    return flag*result;
                }
            }
        }
        return flag*result;
    }

码点与代码单元

2019-10-31

Java中
码点：一个字符在某个编码表中对应的代码值（表示它的数值），Java中指某个字符对应的Unicode值。
代码单元：在UTF-16采用16位来存储基本的多语言级别的某个Unicode字符。此即称为代码单元。
UTF-16中对Unicode中基本的多语言级别采用的是一个码点表示一个字符，而对于其他级别（包含辅助字符…）则是采用两个码点表示一个字符。
例如：
Java中
String greeting=”\uD835\uDD46”;
然后使用
int n=greeting.length();//n==2
返回的就是代码点的数目
而
int m=greeting.codePointCount(0, greeting.length());//n==1
返回的才是真正的Unicode中字符的个数。

题目：
给出一个 32 位的有符号整数，你需要将这个整数中每位上的数字进行反转。

示例 1:

输入: 123

输出: 321

示例 2:

输入: -123

输出: -321

示例 3:

输入: 120

输出: 21

注意:

假设我们的环境只能存储得下 32 位的有符号整数，则其数值范围为 [−2^31, 2^31 − 1]。请根据这个假设，如果反转后整数溢出那么就返回 0。

思路：
1
此题算法非常容易想出，不断对整数取余并将余数放入另一个数字。
此处思路的难处是：正如题目提示的那样，肯定会有数字超出范围的，所以就应该判断数字在何时会超出表示范围。整数：

代码：

class Solution {
    public int reverse(int x) {
        int ans = 0;
        x=Math.abs(x);
        while (x != 0) {
            int pop = x % 10;
            if (ans > Integer.MAX_VALUE / 10 || (ans == Integer.MAX_VALUE / 10 && pop > 7)) /*此处如果上一次的整数大于Integer.MAX_VALUE/10那么这次乘以10肯定会超过范围，如果刚好相等，如果余数还大于7的化加上余数后也会超出范围，Integer.MAX_VALUE是2147483647（2^31-1）*/
                return 0;
            if (ans < Integer.MIN_VALUE / 10 || (ans == Integer.MIN_VALUE / 10 && pop < -8)) /*此处如果上一次的整数小于Integer.MAX_VALUE/10那么这次乘以10肯定比最小还要小（即超出表示范围）。如果刚好相等，如果余数还小于-8的话，加上余数后也会超出范围，Integer.MIN_VALUE是-2147483648（-1*2^31）*/
                return 0;
            ans = ans * 10 + pop;
            x /= 10;
        }
        return ans;
    }
}

2019-10-20

题目：
将字符串 "PAYPALISHIRING" 以Z字形排列成给定的行数：

P   A   H   N
A P L S I I G
Y   I   R

之后从左往右，逐行读取字符： "PAHNAPLSIIGYIR"

实现一个将字符串进行指定行数变换的函数:

string convert(string s, int numRows);

示例 1:

输入: s = “PAYPALISHIRING”, numRows = 3
输出: “PAHNAPLSIIGYIR”

示例 2:

输入: s = “PAYPALISHIRING”, numRows = 4
输出: “PINALSIGYAHRPI”
解释:

P     I    N
A   L S  I G
Y A   H R
P     I

思路：规律是：首先取出第一个字符，如果是第一行与倒数第一行继续以2nrows-1的间隔取下一个字符。如果不是第一与倒数第一行，首先取出第一个字符，然后重复间隔以2norws-1取与间隔为1取下一个字符（注意这个周期的起点字符是一样的）注意取过的字符

代码：

import java.util.Scanner;

public class o0006 {
    public static void main(String[] args) {
        Scanner in=new Scanner(System.in);
        String str=in.nextLine();
        in.close();
        System.out.println(convert(str, 4));
    }
    public static String convert(String s,int numRows) {
        if(numRows==1) {
            return s;
        }else {
            StringBuilder ret=new StringBuilder();
            int n=s.length();//获取字符串的长度
            int cyclen=2*numRows-2;//初始取字符的的下一坐标比上一字符多2*行数-2
            for(int i=0;i<numRows;i++) {
                for(int j=0;j+i<n;j+=cyclen) {//
                    ret.append(s.charAt(i+j));//每行的元素总是两个两个加入到字符串中，起始字符是，第一行与代数第一行除外。
                    if(i!=0&&i!=numRows-1&&j+cyclen-i<n) {
                        ret.append(s.charAt(j+cyclen-i));
                    }
                }
            }
            return ret.toString();
        }
    }
}

 * 对于每个字符串，首先去行首元素，如果是第一行与最后一行就以间隔2*nrows-1的间隔不断取下去，
 * 如果不是这两行每次取两个字符，首先以间隔2*nrows-1取一个字符，然后再以间隔1取一个字符。
 * 如果下表超出长度则结束此次循环，进行下一次循环的字符输入。
