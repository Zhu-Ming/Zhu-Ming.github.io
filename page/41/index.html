#include<stdio.h>

#1006 换个格式输出整数
让我们用字母 B 来表示“百”、字母 S 表示“十”，用 12…n 来表示不为零的个位数字 n（<10），换个格式来输出任一个不超过 3 位的正整数。例如 234 应该被输出为 BBSSS1234，因为它有 2 个“百”、3 个“十”、以及个位的 4。

输入格式：
每个测试输入包含 1 个测试用例，给出正整数 n（<1000）。

输出格式：
每个测试用例的输出占一行，用规定的格式输出 n。

输入样例 1：

234

输出样例 1：

BBSSS1234

输入样例 2：

23

输出样例 2：

SS123

分析：

1因为输入的数字小于1000，所以定义一个数组长度为3的整型数组，存放数字的个位，十位，百位。

2再定义一个足够大的数组存放结果。

3倒叙遍历存放数字的数组，按按百对应下标为2，十位对应的下标为1，个位对应的下标的为0，分别对处理。

4注意一定要在最后将字符数组的最后置为’\0’,否则会输出错误。

代码：

#include<stdio.h>
int main(void){
char result[1000];
int number;
scanf("%d",&number);

int n=0;
int num[3]={0};
while(number){
    num[n++]=number%10;
    number=number/10;
}
int step=0;
for(int i=3;i>=0;i--){
    int time=num[i];
    if(time==0){
        continue;
    }
    if(i==2)
        for(int j=0;j<time;j++){
        result[step++]='B';
    }
    if(i==1)
        for(int j=0;j<time;j++){
        result[step++]='S';
    }
    if(i==0){
        for(int j=0;j<time;j++){
        result[step++]=j+'1';
        }
    }

}
result[step]='\0';
printf("%s",result);
return 0;
}

卡拉兹(Callatz)猜想已经在1001中给出了描述。在这个题目里，情况稍微有些复杂。

当我们验证卡拉兹猜想的时候，为了避免重复计算，可以记录下递推过程中遇到的每一个数。例如对 n=3 进行验证的时候，我们需要计算 3、5、8、4、2、1，则当我们对 n=5、8、4、2 进行验证的时候，就可以直接判定卡拉兹猜想的真伪，而不需要重复计算，因为这 4 个数已经在验证3的时候遇到过了，我们称 5、8、4、2 是被 3“覆盖”的数。我们称一个数列中的某个数 n 为“关键数”，如果 n 不能被数列中的其他数字所覆盖。

现在给定一系列待验证的数字，我们只需要验证其中的几个关键数，就可以不必再重复验证余下的数字。你的任务就是找出这些关键数字，并按从大到小的顺序输出它们。

输入格式：
每个测试输入包含 1 个测试用例，第 1 行给出一个正整数 K (<100)，第 2 行给出 K 个互不相同的待验证的正整数 n (1<n≤100)的值，数字间用空格隔开。

输出格式：
每个测试用例的输出占一行，按从大到小的顺序输出关键数字。数字间用 1 个空格隔开，但一行中最后一个数字后没有空格。

输入样例：

6
3 5 6 7 8 11

输出样例：

7 6

分析：

1定义一个数组存储输入的数字。

2遍历数组的每一个数字，对它们每个数字求其覆盖的数字，对被覆盖的每个数字一一与输入的数组对比，如果有相同的就将处于该位置的数字置为1，因为题目中提到数字在（1<n<=100）之间，所以便利该数组时可以遇到一就跳过去。

3最后数组中只剩下不能被数组中的其他元素所覆盖的数字了，这就是要求的关键数的数组。

4对这个数组进行排序，直到第一个为1的下标。

5按要求输出下标前面的数字。

代码：

#include<stdio.h>
#include<stdlib.h>
//此处是比较函数，用于对处理后的数组排序。
int compare(const void *a,const void *b){
if(*(int *)a>*(int *)b)
return -1;
if(*(int *)a<*(int *)b)
return 1;
return 0;
}
int main(void){
int number;
scanf("%d",&number);
int num[number];
int i=0;
while(i<number){ 
    scanf("%d",&num[i++]);
}
for(i=0;i<number;i++){
    int n=num[i];
    if(n==1){
        continue;
    }
    while(1){
     if(n==1){
        break;
    }
    if(n%2==0){
        n=n/2;
    }else{
        n=(3*n+1)/2;
    }
    for(int m=0;m<number;m++){
        if(n==num[m]){
            num[m]=1;// 因为数组的数字各不相同，找到被覆盖的即可跳出循环
            break;
        }
      }
    }
}
qsort(num,sizeof(num)/sizeof(int),sizeof(int),compare);
int flag=1;
for(int i=0;i<number;i++）{
    if(num[i]==1)break;
    if(flag){
        flag=0;
        printf("%d",num[i]);
    }else{
        printf(" %d",num[i]);
    }
}

}
