LeetCode-23合并多个链表

2019-11-14

题目：

合并 k 个排序链表，返回合并后的排序链表。请分析和描述算法的复杂度。
示例:
输入: [   1->4->5,   1->3->4,   2->6 ] 输出: 1->1->2->3->4->4->5->6

思路一（自写）：递归：效率极低，战胜15%，运行时间：340ms

每次取出lists的所有链表的第一个元素，找出最小的那个，然后用最小值的那个结点，指着这个结点的指针指向他所在结点后的链表与其他结点继续此操作的返回值。

代码：

public ListNode mergeKLists(ListNode[] lists) {
    int minval=Integer.MAX_VALUE;
    int minIndex=0;
    ListNode head=null;
    for(int i=0;i<lists.length;i++) {
        if(lists[i]==null) {
            continue;
        }
        if(lists[i].val<=minval) {
            minval=lists[i].val;
            minIndex=i;
        }
    }
    if(minval==Integer.MAX_VALUE)
        return null;
    head=lists[minIndex];
    lists[minIndex]=lists[minIndex].next;
    head.next=mergeKLists(lists);
    return head;
}

思路二：归并：效率高，战胜100%，运行时间：2ms

将lists不断合并左右元素，然后放入左半边lists，然后继续在左半边继续合并左右元素操作，直到right到达0处。两个链表的合并算法自己想。

代码：

public ListNode 自写mergeKLists(ListNode []lists) {
    if (lists==null||lists.length<=0)
        return null;
    ListNode head=new ListNode(0);
    int left=0;
    int right=lists.length-1;
    while(right>0) {
        while(left<right) {
            head.next=自写merge(lists[left],lists[right]);
            left++;
            right--;
        }
        left=0;
    }
    return head.next;
}
private ListNode 自写merge(ListNode l1, ListNode l2) {
    if(l1==null)//如果有一个为null，直接返回另一个。
        return l2;
    if(l2==null)
        return l1;
    ListNode head=new ListNode(0);//建一个头结点不做返回值一部分
    ListNode currNode=head;//定义一个零时工。
    while(l1!=null&l2!=null) {
        if(l1.val<l2.val) {
            currNode.next=l1;
            l1=l1.next;
        }else {
            currNode.next=l2;
            l2=l2.next;
        }
        currNode=currNode.next;
    }
    currNode.next=l1==null?l2:l1;//如果有链表没有结束，直接将其加到后面。
    return head.next;
}

题目：

给出 n 代表生成括号的对数，请你写出一个函数，使其能够生成所有可能的并且有效的括号组合。

例如，给出 n = 3，生成结果为：

[
  "((()))",
  "(()())",
  "(())()",
  "()(())",
  "()()()"
]

思路：回溯算法。

方法一：回溯算法（深度优先遍历）
如果完成一件事情有很多种方法，并且每一种方法分成若干步骤，那多半就可以使用“回溯”算法完成。

“回溯”算法的基本思想是“尝试搜索”，一条路如果走不通（不能得到想要的结果），就回到上一个“路口”，尝试走另一条路。

因此，“回溯”算法的时间复杂度一般不低。如果能提前分析出，走这一条路并不能得到想要的结果，可以跳过这个分支，这一步操作叫“剪枝”。

做“回溯”算法问题的基本套路是：

1、使用题目中给出的示例，画树形结构图，以便分析出递归结构；

一般来说，树形图不用画完，就能够分析出递归结构和解题思路。

2、分析一个结点可以产生枝叶的条件、递归到哪里终止、是否可以剪枝、符合题意的结果在什么地方出现（可能在叶子结点，也可能在中间的结点）；

3、完成以上两步以后，就要编写代码实现上述分析的过程，使用代码在画出的树形结构上搜索符合题意的结果。

在树形结构上搜索结果集，使用的方法是执行一次“深度优先遍历”。在遍历的过程中，可能需要使用“状态变量”。

（“广度优先遍历”当然也是可以的，请参考方法二。）

我们以 n = 2 为例，画树形结构图。

画图以后，可以分析出的结论：

左右都有可以使用的括号数量，即严格大于 00 的时候，才产生分支；

左边不受右边的限制，它只受自己的约束；

右边除了自己的限制以外，还收到左边的限制，即：右边剩余可以使用的括号数量一定得在严格大于左边剩余的数量的时候，才可以“节外生枝”；

在左边和右边剩余的括号数都等于 00 的时候结算。

class Solution {
    public List<String> generateParenthesis(int n) {
         List<String>list=new ArrayList<String>();
         if(n==0)
             return list;
         String str="";
         def(n,n,str,list);
         return list;
        }
    public static void def(int left,int right,String str,List<String>list) {
        if(left==0&&right==0) {
            list.add(str);
            return ;
        }
        if(left>0)
        def(left-1,right,str+"(",list);
        if(right>0&&right>left)
        def(left,right-1,str+")",list);
        return ;
    }
}

LeetCode-21合并两个有序链表

2019-11-12

题目：

将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

示例：

输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4

思路1：

新建一个节点，作为头节点，不作为返回值的一部分。然后再定义一个零时工，用于不断扩大头节点的长度，采
用归并算法，归并两个有序数组。然后返回头节点的下一个节点就可以了。注意：两个有序链表可能为空。还有
就是归并过程中不必再创建新的节点，直接使用原来链表的结点，将其加到返回值链表中就可以了。创建新的结点会浪费大量的时间。

代码：

public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
ListNode headNode=new ListNode(0);
headNode.next=null;//此处一定要将头节点的下一个节点设为null，不然当两个链表都为空时，会显示异常。
ListNode p=headNode;
int val1=0;
int val2=0;
while(l1!=null&&l2!=null) {
    val1=l1.val;
    val2=l2.val;
    if(val1<val2) {
        p.next=l1;
        l1=l1.next;
    }else {
        p.next=l2;
        l2=l2.next;
    }
    p=p.next;
}
while(l1!=null) {
    p.next=l1;
    l1=l1.next;
    p=p.next;
}
while(l2!=null) {
    p.next=l2;
    l2=l2.next;
    p=p.next;
}
return headNode.next;
}

思路二：递归算法实现。

将两个链表的首节点中，小的那一个作为首节点返回，然后将被选中的那个节点的后一个节点以及它后面的节
点当成新的链表继续与另外一个链表归并。重复次过程直到，某一个节点为null，此时直接将另一个链表加到
结果链表中返回就可以了。


也是将链表归并的过程看成了1选出两个链表中首节点小的那个作为返回链表的首节点，然后剩余部分再归并的
过程。

代码：

public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    if (null == l1) return l2;
    if (null == l2) return l1;
    if (l1.val < l2.val) {
        l1.next = mergeTwoLists(l1.next, l2);
        return l1;
    } else {
        l2.next = mergeTwoLists(l1, l2.next);
        return l2;
    }
}

LeetCode-20有效的括号

2019-11-11

题目：

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: "()"
输出: true
示例 2:

输入: "()[]{}"
输出: true
示例 3:

输入: "(]"
输出: false
示例 4:

输入: "([)]"
输出: false
示例 5:

输入: "{[]}"
输出: true

思路：
利用堆栈读取括号遇到左括号’(‘,’{‘,’[‘将其进栈，遇到右括号先取栈顶元素看看是不是与当前元素匹配的括号，不是则说明，括号不匹配，立即输出false，如果匹配，就先出栈，然后继续读取下一个字符。

代码:

public boolean isValid(String s) {
   if(s == null || s.length() % 2 == 1) return false;
    if(s.length()==0) {
        return true;
    }
    Stack<Character> stack=new Stack<>();
    for(int i=0;i<s.length();i++) {
        char ch=s.charAt(i);
        if(ch=='['||ch=='('||ch=='{') {
            stack.push(ch);
        }else {
            if(stack.isEmpty()) {
                return false;
            }
            if(ch==']'&&stack.elementAt(stack.size()-1)=='[') {
                stack.pop();
            }else if(ch==')'&&stack.elementAt(stack.size()-1)=='(')
            {
                stack.pop();
            }else if(ch=='}'&&stack.elementAt(stack.size()-1)=='{') {
                stack.pop();
            }else {
                return false;
            }
        }
    }
    return stack.isEmpty();
}

思路：

自己实现栈然后重复以上操作，基本思路一样。

LeetCode-19删除链表倒数第N个节点

2019-11-10

题目：

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

示例：

给定一个链表: 1->2->3->4->5, 和 n = 2.

当删除了倒数第二个节点后，链表变为 1->2->3->5.
说明：

给定的 n 保证是有效的。

思路：

1计算链表的长度
2找到长度减去n的节点位置
3交换节点
注意：如果n为链表长度时会有错误
需要补充一下此时直接返回头节点的下一个节点就可以了

代码：

public ListNode removeNthFromEnds(ListNode head, int n) {
        ListNode node=head;
        int len=0;
        while(node!=null) {
            len++;
            node=node.next;
        }
        if(len==n) {
            return head.next;
        }
        node=head;
        for(int i=1;i<len-n;i++) {
            node=node.next;
        }
        ListNode aNode=node.next;
        node.next=aNode.next;
        return head;
    }

思路2：递归到达最底层后开始计数，当数字达到n时返回此节点的下一个节点给上一个节点。（佩服佩服，果然时高手，佩服）

代码：

private int time;
public ListNode removeNthFromEnd(ListNode head,int n) {
    if(head==null) {//当head为null时表明已经达到最后一个节点的下一个指针
        time=0;
        return null;
    }
    head.next=removeNthFromEnd(head.next,n);//要保证原链表的结构不变
    time++;//买次递归返回的时候就加1
    if(time==n) {//当达到第n个的时候返回当前节点的下一个节点
        return head.next;
    }
    return head;//否则返回当前节点保证原链表的结构不变
}

LeetCode-18四数之和

2019-11-09

题目：

给定一个包含 n 个整数的数组 nums 和一个目标值 target，判断 nums 中是否存在四个元素 a，b，c 和 
d ，使得 a + b + c+ d 的值与 target 相等？找出所有满足条件且不重复的四元组。

 注意：
 答案中不可以包含重复的四元组。

 示例：
 给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。

 满足要求的四元组集合为： [ [-1, 0, 0, 1], [-2, -1, 1, 2], [-2, 0, 0, 2] ]

 来源：力扣（LeetCode） 链接：https://leetcode-cn.com/problems/4sum
 著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

思路：
双指针选定法，首先对数组进行排序，然后定义一个指向最左与最右的指针，然后从这两个指针中间选取两个值构造四数数组，看看他们的和合不合适，合适就加到数组列表中，不合适就变换中间的两个值。注意此时可以用一些剪枝方法出去不合适的四个数字。

代码：

    public List<List<Integer>> fourSum(int[] nums, int target) {
        List<List<Integer>> list = new ArrayList<>();
        if (nums.length < 4) {//剪枝方法
            return list;
        } else {
            Arrays.sort(nums);
            int l = 0, r = nums.length - 1, lr = 0, rl = 0;
            for (l = 0; l < nums.length - 3; l++) {
                if (l > 0 && nums[l] == nums[l - 1]) {
                    continue;
                }
                for (r = nums.length - 1; r >= l +3; r--) {
                    if (r < nums.length - 1 && nums[r] == nums[r + 1]) {//剪枝方法
                        continue;
                    }
                    lr = l + 1;//开始指针的右边一个指针
                    rl = r - 1;//最右边一个指针的左边一个指针
                    while (lr < rl) {
                        int value = nums[l] + nums[r] + nums[lr] + nums[rl];
                        if (value == target) {
                            list.add(Arrays.asList(nums[l], nums[lr], nums[rl], nums[r]));
                            while (lr < rl && nums[lr] == nums[lr + 1])//剪枝方法
                                lr++;
                            while (lr < rl && nums[rl] == nums[rl - 1])//剪枝方法
                                rl--;
                            lr++;
                            rl--;
                        } else if (value > target) {
                            rl--;
                        } else {
                            lr++;
                        }
                    }
                }
            }
        }
        return list;


}

public List<List<Integer>> fourSum(int []nums,int target){
    int len=nums.length;
    List<List<Integer>>list=new ArrayList<>();
    if(len<4) 
    {
        return list;
    }
    int value;
    Arrays.sort(nums);
    for(int i=0;i<len-3;i++) 
    {
        if(i>0&&nums[i-1]==nums[i])//如果这一个跟上一个一样就没必要再去取这一个了
        {
            continue;
        }
        value=nums[i]+nums[i+1]+nums[i+2]+nums[i+3];//如果以nums[i]开头的数组的最小的和都大于target，说明i以后的数字之和都大于target。
        if(value>target)
        {
            break;
        }
        value=nums[i]+nums[len-1]+nums[len-2]+nums[len-3];
        if(value<target)//如果以nums[i]开头的数组中的最大值都小于target，则说明i所有的数组之和都小于target
        {
            continue;
        }
        for(int j=i+1;j<len-2;j++)
        {
            if(j>i+1&&nums[j]==nums[j-1])//意义同上一层
            {
                continue;
            }
            value=nums[i]+nums[j]+nums[j+1]+nums[j+2];
            if(value>target) {//意义同上一层
                break;
            }
            value=nums[i]+nums[j]+nums[len-1]+nums[len-2];
            if(value<target) {//意义同上一层
                continue;
            }
            int l=j+1;
            int r=len-1;
            while(r>l) {
                value=nums[i]+nums[j]+nums[l]+nums[r];
                if(value==target) {
                    list.add(Arrays.asList(nums[i],nums[j],nums[l],nums[r]));
                    while(l<r&&nums[l]==nums[l+1])l++;
                    while(l<r&&nums[r]==nums[r-1])r--;
                    l++;
                    r--;
                }else if(value>target) {
                    r--;
                }else {
                    l++;
                }
            }
        }
    }
    return list;
}
