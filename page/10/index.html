题目：

给定一个字符串，找出字符串中的最长回文子字符串。你可以假设s的最大长度为1000.

实例：

输入：“babad”

输出：“bab”

注意：aba也是一个有效答案。

实例：

输入:“cbbd”

输出：“bb”

思路：

一：

三重循环，遍历字符串，在下标为i到j之间判断其是否是回文字符串，是与目前最大长度相比，大的就赋值，小的舍弃。

二：

循环遍历字符串，检查每个字母的左右是否是回文字符串，以及它自身是否与其他字符构成回文字符串（比如“abba”）有记录下他们的起始位置，并与目前最大的长度相比，而后同上。

代码：

import java.util.ArrayList;

public class o0005 {
    public static void main(String[] args) {
        System.out.println(Solution_2("babad").toString());
    }

    public static ArrayList<String> Solution_1(String str) {
        ArrayList<String> list=new ArrayList<String>();
        int max=0;
        for(int i=0;i<str.length();i++) {
            for(int j=str.length();j>i;j--) {
                int flag=0;
                    for(int m=0,n=str.substring(i, j).length()-1;m<n;m++,n--) {
                        if(str.substring(i, j).charAt(m)!=str.substring(i, j).charAt(n)) {
                            flag=1;
                            break;
                        }
                    }
                    if(str.substring(i, j).length()>=max&&flag==0) {
                        if(str.substring(i, j).length()>max) {
                            list.clear();
                        }
                        max=str.substring(i, j).length();
                        list.add(str.substring(i,j));
                    }
            }
        }
        return list;
    }
    public static String Solution_2(String str) {
        if(str==null||str.length()<1) {
            return "";
        }
        int start=0,end=0;
        for(int i=0;i<str.length();i++) {
            int len1=expandAroundCenter(str,i,i);//首先检查该字符两侧有没有反转的字符
            int len2=expandAroundCenter(str,i,i+1);//然后检测该字符是否与周围的字符反转
            int len=Math.max(len1, len2);
            if(len>end-start) {//记录最大值处的字符串的位置
                start=i-(len-1)/2;
                end=i+len/2;
            }
        }
        return str.substring(start,end+1);
    }
    private static int expandAroundCenter(String str, int left, int right) {
        int L=left,R=right;
        while(L>=0&&R<str.length()&&str.charAt(L)==str.charAt(R)) {
            R++;
            L--;
        }
        return R-L-1;
    }
}

题目：

给定两个大小位n,m的两个有序数组nums1,nums2.请找出这两个有序数组的中位数，要求时间复杂度位O(log(m+n))。

实例:

nums1[1,3]

nums2[2]

中位数是：2.0

实例：
nums1[1,2]

nums2[3,4]

中位数是：（2+3）/2=2.5

思路：

一：

归并排序，查找中间的两个数字或一个数字，并求它们或它的平均值。

二：
理解不了日后补上

代码：

public class o0004 {
    public static void main(String[] args) {
        int []a=new int[] {1,2};
        int []b=new int[] {3,5};
        System.out.println(Solution_2(a,b));
    }
    public static double Solution_1(int[] nums1,int []nums2) {
        double res=0.0;
        int n=nums1.length,m=nums2.length,i=0,j=0,curr=0;
        int[] result=new int[n+m];
        while(i<n&&j<m) {
            if(nums1[i]<nums2[j]) {
                result[curr]=nums1[i];
                i++;
            }else{
                result[curr]=nums2[j];
                j++;
            }
            curr++;
        }
            while(i<n) {
                result[curr]=nums1[i];
                curr++;
                i++;
            }
            while(j<m) {
                result[curr]=nums2[j];
                curr++;
                j++;
            }
        if(curr%2==0) {
            res=(result[curr/2+1-1]*1.0+result[curr/2-1]*1.0)/2;
        }else {
            res=result[curr/2+1-1]*1.0;//加1是后面的一个数，减去1代表数组下标。
        }
        return res;
    }
    public static double Solution_2(int[] A,int []B) {
        int m=A.length;
        int n=B.length;
        if(m>n) {//确保m<=n
            int[] temp=A;A=B;B=temp;
            int k=m;m=n;n=k;
        }
        int iMin=0,iMax=m,halfLen=(m+n+1)/2;
        while(iMin<=iMax) {
            int i=(iMin+iMax)/2;
            int j=halfLen-i;
            if(i<iMax&&B[j-1]>A[i]) {
                iMin=iMin+1;//i太小了
            }else if(i>iMin&&A[i-1]>B[j]) {
                iMax=iMax-1;//i太大了
            }else {//i是合适的
                int maxLeft=0;
                if(i==0) {
                    maxLeft=B[j-1];
                }else if(j==0) {
                    maxLeft=A[i-1];
                }else {
                    maxLeft=Math.max(A[i-1],B[j-1]);
                }
                if((n+m)%2==1) {
                    return maxLeft;
                }
                int minRight=0;
                if(i==m) {
                    minRight=B[j];
                }else if(j==n) {
                    minRight=A[i];
                }else {
                    minRight=Math.min(A[i], B[j]);
                }
                return (maxLeft+minRight)/2.0;
            }
        }
        return 0.0;
    }
}

LeetCodeo0003

2019-10-20
LeetCode

实现语言：Java

题目：

给定一个字符串，找出不含重复字符的最长字串的长度。

实例：

给定字符串“abcabcbb”,没有重复字符的最长字串是“abc”，那么长度就是3.

给定字符串“bbbbb”,最长的字串就是”b”,长度是1.

给定”pwwkew”,最长字串是“wke”,长度是3。

注意：答案必须是一个字串，“pwke”是子序列，而不是子字符串。

思路：

大体上就是一种思路，只是使用的工具不同罢了。

一：

将输入的字符串一个一个存入集合中，如果集合中已经含有有某个字符了，说明在此处已经达到了一个符合题目要求的字串，如果它的长度比上一个大，就将它的长度赋值给max.只要已经找到重复的字符了，就应该将集合清空，并将这个重复的字符输入到集合中，用于下一次的计算。如果字符集合中没有这个字符也应该输入字符集合，用于下一次的计算。

二：

首先定义两个坐标一个用于定位当前字符的位置，另一个用于定位重复字符在字符串中的位置，然后用每个长度与最大值比较（并决定是否改变最大值）。

三：

定义一个map用于存放字符与其相应的位置，然后边加入，边查找有没有重复的字符，如果有，取出当前重复字符的位置（与上一个重复字符的位置比较，防止其坐标比上一个重复的字符坐标小）。然后求长度并用当前对打长度与目前所求长度比对。加入这个字符，与对应的位置。

代码：

import java.util.*;

public class o0003 {
    public static void main(String[] args) {
        System.out.println(Solution_3("pwwkew"));
    }
    /**
     * @param sub 输入的字符串
     * @param max 用于存储最长字串的长度
     * @return max 最长字串的长度
     */
    public static int Solution_1(String sub) {
        int max=0;
        Set<Character> coll=new HashSet<Character>();
        for(int i=0;i<sub.length();i++) {
            if(coll.contains(sub.charAt(i))) {//就算此处集合中已经有了这个字符，也应该清空集合，加入这个字符以便下一次的计算。
                if(coll.size()>max) {
                    max=coll.size();
                }
                coll.clear();//如果有重复的就清空集合
            }
            coll.add(sub.charAt(i));
        }
        return max;
    }
    /**
     * @param i 用于记录重复字符的位置。
     * @param sub
     * @return
     */
    public static int Solution_2(String sub) {
        int max=0,len=sub.length();
        int i=0,j=0;
        Set <Character> set=new HashSet<>();
        while(i<len&&j<len) {
            if(!set.contains(sub.charAt(j))) {//如果集合中没有这个字符，就将其加入集合，并将集合的长度与目前为止的最大长度比较，然后决定是否改变它。
                set.add(sub.charAt(j++));//首先将其加入集合中。
                max=Math.max(max, j-i);//比较。
            }else {
                set.remove(sub.charAt(i++));//如果有这个字符了，就一直删除字符，直到找到重复的那个字符,重复的那个字符也会被删除。
            }
        }
        return max;
    }
    public static int Solution_3(String sub) {
        int len=sub.length(),max=0;
        Map<Character,Integer>map=new HashMap<>();
        for(int i=0,j=0;j<len;j++) {
            if(map.containsKey(sub.charAt(j))) {
                i=Math.max(i, map.get(sub.charAt(j)));//i用来存储当前已有字符在字符串中的位置。为什么要与上一个i比较呢？因为上一个i代表的是上一个重复的字符在字符串中的位置，而这一个字的位置可能在上一个字符的后面所以因该取他们的最大值。
            }
            max=Math.max(max, j+1-i);//j+1代表当前字符在字符串中的位置，i代表上一个重复字符在字符串中的位置。
            map.put(sub.charAt(j), j+1);//将字符放入hash表中，如果已经有就覆盖它。
        }
        return max;
    }
}
